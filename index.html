Actionkin Scale

Scale is a web-based project management platform designed for the client ActionKin to streamline team collaboration, task tracking, and progress monitoring.

I worked primarily on the database using Drizzle ORM, achieving close to 100% code coverage with Jest for integration tests and implementing Selenium UI tests to ensure end-to-end reliability.
